#  Bank Marketing Campaign Analysis

##  Project Overview
This project presents a complete data analysis workflow on a bank marketing dataset. It includes data cleaning, preprocessing, exploratory data analysis (EDA), and interactive dashboard development to uncover key factors influencing term deposit subscription. The goal is to extract actionable business insights and support data-driven marketing strategies.

---

##  Tools Used
* **Excel:** For data cleaning, handling missing values, and initial exploration.
* **Power BI:** For data modeling, DAX calculations, and creating an interactive dashboard.

---

##  Key Insights (From Dashboard)
* **Top Performing Segment:** Management professionals and married individuals show the highest count of successful campaign outcomes.
* **Education Impact:** Customers with a **Secondary** education level represent the largest group of successful conversions (**47.02%**).
* **Communication Channel:** Cellular contact is the most effective method, showing a significantly higher success rate.
* **Campaign Timing:** There is a notable peak in call durations during the month of **May**, suggesting high engagement periods.
* **Efficiency Metrics:** The average call duration is approximately **263.96**, with a total of **4,521** calls analyzed.
